# Code Minimalism

Personal Blog.

### Building
To run the image locally
```
bundle exec jekyll serve
```

and then access it on local host port 4000 
`http://127.0.0.1:4000/`

To publish a change
```
bundle exec jekyll build
``` 
And then push the changes under the `_site` dir

### Template
The template used is from [hydejack](https://hydejack.com/) and is forked from the repo [hydecorp/hydejack](https://github.com/hydecorp/hydejack)
